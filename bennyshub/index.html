NARBE House | Benny's Accessibility Hub
This is a hub for 2-button switch access mapped to spacebar and return and/or your mouse/headtracking/eyetracking.
Below are tools and games that will be populated over time that are free to access and use. Use spacebar to scan and return to select. Hold spacebar for 3s to scan backwards. Mouse clicks work too. Holding down return will open the pause menu on games.
